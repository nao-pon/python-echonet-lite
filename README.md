# Wi-SUN Gateway for Docker

## Changes

- BP35A1に対応
- Docker化
- WebUIの削除
- Display、ボタン等Wi-SUNアダプタ以外のHWの削除

## Installation

```
$ docker compose up -d --build
```

## Configuration

config.ini.defaultをconfig.iniにリネームし設定値を書き換える
